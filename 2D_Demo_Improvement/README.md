How to Play  
Open this folder as a Unity project.  
Open Assets/_Scenes/MainMenu.unity.  
Press Play.  
Click New Game to start the game.  
Click Level Select to choose Level 1, Level 2, or Level 3.  
Move the spaceship with Arrow Keys or WASD.  
Shoot enemies with the left mouse button.  
Press Esc to pause the game.  
Destroy all enemies to complete the level.  
Avoid enemy attacks. Each hit removes 1 life.  
When lives reach 0, the game is over.
