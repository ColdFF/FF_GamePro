# Space Blaster: Survival Mission

## Unity Version

Unity 2022.3.62f3 LTS

## Scene

Open:

`Assets/_Scenes/MainMenu.unity`

## How to Play

1. Open this folder as a Unity project.
2. Open `Assets/_Scenes/MainMenu.unity`.
3. Press Play.
4. Click **New Game** to start the game.
5. Click **Level Select** to choose Level 1, Level 2, or Level 3.
6. Move the spaceship with Arrow Keys or WASD.
7. Shoot enemies with the left mouse button.
8. Press Esc to pause the game.
9. Destroy all enemies to complete the level.
10. Avoid enemy attacks. Each hit removes 1 life.
11. When lives reach 0, the game is over.
