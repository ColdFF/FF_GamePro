# Interactive Solar System for Kids

## Unity Version

Unity 2022.3.62f3 LTS

## Scene

Open:

Assets/Scenes/SampleScene.unity

## How to Play

1. Open this folder as a Unity project.
2. Open `Assets/Scenes/SampleScene.unity`.
3. Press Play.
4. Click Earth, Moon, or Mars to focus the camera.
5. Press Back to return to the main view.

## Features

- Sun, Earth, Moon, and Mars
- Planet and moon orbit motion
- Clickable Earth, Moon, and Mars
- Camera movement
- Back button
- Visual scale response
- Space texture skybox
- Audio feedback
- Mini map
